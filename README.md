# ConfidUp

ConfidUp is a WordPress website for a digital marketing agency specializing in:

- **SEO** (Search Engine Optimization)
- **AEO** (Answer Engine Optimization)
- **GEO** (Generative Engine Optimization)

## Repository

https://github.com/Confidup/confidUp-com

## Tech Stack

- WordPress

## Getting Started

1. Clone the repository
2. Set up a local WordPress environment (e.g. LocalWP, XAMPP, or Docker)
3. Import the database and configure `wp-config.php`
4. Install and activate required plugins and theme
